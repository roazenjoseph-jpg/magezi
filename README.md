# Anesis Air Solutions Website

Professional website for Anesis Air Solutions - Air Conditioning & Ventilation Services in Kampala, Uganda

## 🚀 Features
- ✅ Responsive Design (Desktop, Tablet, Mobile)
- ✅ Animated Hamburger Menu
- ✅ Hero Background Slideshow
- ✅ Before / After Gallery with synchronized transitions
- ✅ Testimonial Carousel
- ✅ Product Cards with scroll animations
- ✅ Working Contact Form
- ✅ Floating WhatsApp Button
- ✅ Google Maps Integration
- ✅ Service Pricing Packages

## 📁 Project Structure
```
github-website/
├── index.html              # Main HTML file
├── css/
│   └── style.css           # Stylesheet
├── js/
│   └── main.js             # JavaScript functionality
├── img/                    # Place all your images here
├── README.md               # This file
└── .gitignore              # Git ignore rules
```

## 🛠️ Installation
1. Clone this repository
2. Place all your project images into the `/img` folder
3. Open `index.html` in your browser
4. Upload to any web hosting provider

## 📝 Customization
- Edit `css/style.css` for design changes
- Modify `js/main.js` for functionality changes
- Update content directly in `index.html`

## 🌐 Live Demo
Website is designed to run on any standard web server. No dependencies or build process required.
